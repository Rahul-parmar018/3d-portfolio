# Post-Launch AI/ML Engineering Roadmap

This document outlines the planned project iterations to evolve this portfolio from an application framework into a deep AI/ML engineering showcase.

---

## 📅 Month 1: PyTorch & Technical Writing
- **Project:** Custom PyTorch model training run (e.g., MNIST/CIFAR-10 classification or simple transformer blocks).
- **Deliverables:**
  - Colab/Jupyter Notebook detailing data loading, optimizer parameters, learning rate schedules, and loss evaluation curves.
  - Integration of experiment logs using Weights & Biases or TensorBoard.
- **Article:** Publish a technical deep-dive article on Medium, Dev.to, or a personal sub-blog detailing the PyTorch training results.

---

## 📅 Month 2: Hugging Face Spaces & RAG Case Study
- **Project:** A production-grade Retrieval-Augmented Generation (RAG) system with retrieval evaluation.
- **Deliverables:**
  - Deploy a model interface on a **Hugging Face Space**.
  - Document vector database choice (ChromaDB/Pinecone), chunking strategy, evaluation metrics (RAGAS framework), and query latencies.

---

## 📅 Month 3: Computer Vision & Benchmark Performance
- **Project:** Applied Computer Vision pipeline (e.g., object detection, segmentation, or OCR parser refinement).
- **Deliverables:**
  - Real-time video/image processing inference pipeline.
  - Published benchmarks (inference latency, model footprint size, accuracy metrics on target datasets).

---

## 📅 Month 4: MLOps & CI/CD Pipelines
- **Project:** Autonomous model deployment pipeline.
- **Deliverables:**
  - Automated deployment pipeline triggered upon model weights update.
  - Integration of monitoring metrics, model validation gates, and serverless edge functions.
